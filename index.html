CCTV Footage based Intelligence Pedestrian Crossing System

Introduction


CCTV Footage based Intelligence Pedestrian Crossing System is a research based project which came up with a solution for current pedestrians crossing system within Srilanka. This system identifies the pedestrians on the pedestrian crossing using CCTV camera feeds placed on the crossings. With the use of identified pedestrian count and vehicle density the system calculates the green time for both pedestrians and vehicles on eeach iteration. The main advantage of this system is, its allowing the vehicles to continue on the road within situations no pedestrians on the crossing. So its saving the vehicle drivers time unless keeping them on the crossing invainly.
